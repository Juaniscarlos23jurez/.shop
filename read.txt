## **Paleta de Colores**

**Color Principal:**

- Verde esmeralda (`#22c55e`) como color de marca - lo uso para botones principales, navegación activa y elementos de éxito


**Colores de Fondo:**

- Gris muy claro (`#f8fafc`) como fondo general de la app
- Blanco puro para las cards y contenedores principales
- Fondos de color suave (blue-50, green-50, purple-50) para iconos y estados


**Colores de Texto:**

- Gris muy oscuro (`#0f172a`) para títulos y texto principal
- Gris medio (`#64748b`) para texto secundario y descripciones
- Gris claro (`#94a3b8`) para placeholders y metadatos


**Colores Semánticos:**

- Azul para métricas y analytics
- Verde para ventas y conversiones
- Púrpura para comentarios y social
- Naranja para alertas y tendencias


## **Tipografía**

**Jerarquía:**

- Títulos principales: 30px, peso bold (900)
- Subtítulos: 24px y 20px, peso bold
- Texto normal: 14px, peso medium (500)
- Texto pequeño: 12px para metadatos y labels


**Estilo:**

- Fuente Inter (moderna y legible)
- Texto suavizado (antialiased)
- Espaciado de letras normal
- Altura de línea generosa para legibilidad


## ️ **Estructura de Layout**

**Contenedor Principal:**

- Fondo gris claro en toda la pantalla
- Padding horizontal consistente de 16px
- Espaciado vertical de 24px entre secciones
- Padding bottom de 96px para la navegación flotante


**Sistema de Cards:**

- Fondo blanco con esquinas muy redondeadas (16px)
- Sombras sutiles con transparencia baja
- Bordes casi imperceptibles en gris claro
- Padding interno de 20-24px
